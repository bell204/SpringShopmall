<h1>팀 프로젝트명: FOREST(예술가 후원 웹사이트)</h1>

<h3>개발기간</h3>

4.9 ~ 5.9

<h3>개발 목표</h3>

스프링을 이용하여 독자가 신진 예술가를 후원하고, 예술가는 창작에 전념할 수 있는 웹사이트 개발

<h3>개발 환경</h3>

운영체제- Window8

사용 언어- Java, Javascript, Html, css

라이브러리, 프레임워크- Jquery, Spring

IDE- 이클립스
